Automative Repair Database


Develop an automotive repair information system to capture repair orders and print invoices for the repair of vehicles.

  • Conceptual data modeling (entities, relationships, attributes, keys)

  • Enhanced Entity-Relationship (EER) diagram design and refinement

  • Logical data modeling and normalization to BCNF

  • Defining and enforcing integrity constraints
    • primary keys
    • foreign keys
    • alternate keys
    • entity integrity

  • Translating logical models into PostgreSQL schema using SQL

  • Writing SQL queries for application requirements and sample transactions

  • Implementing referential integrity and general constraints in PostgreSQL

  • Data insertion and test dataset creation for validation

  • Using pg_dump for database backup and submission

  • Technical documentation with diagrams, SQL code, outputs, and conclusions
